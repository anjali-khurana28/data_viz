Crime Rate in America (2015-2016)

Directions

  1. Select the year to view: Aggregated crime rate heat map.
  2. Hover state: Hover to view aggregated rate in the state.
  3. Select state level detail: Select the Explore button. Select a state to view streamgraph.
  4. For comparing two states: Select the Comparison Button. Press State 1 button and select a state on the map. Similar steps for state 2
